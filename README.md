# 📝 TodoApp

Basit ve kullanışlı bir yapılacaklar listesi (Todo List) uygulamasıdır.  
Kullanıcılar todo ekleyebilir, silebilir ve tüm listeyi temizleyebilir. Veriler tarayıcının **Local Storage** özelliği ile saklanır.

## 🚀 Özellikler

- ➕ Todo ekleme
- ❌ Tek tek todo silme
- 🗑️ Tüm todoları temizleme
- 🔍 Todo filtreleme (arama)
- 💾 Verileri Local Storage’da saklama

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3 (Bootstrap 4)
- JavaScript (Vanilla JS)
- Local Storage API
