#🚀 JavaScript Projects Collection
Bu depo, JavaScript dilinin temel prensiplerinden başlayarak; Asenkron Programlama, Nesne Yönelimli Programlama (OOP), API Entegrasyonları ve State Management konularında kendimi geliştirmek amacıyla hazırladığım 9 farklı projeyi içermektedir.

📁 Proje Listesi ve Teknik Detaylar
##1. Todo List App 📝
Odak: DOM Manipülasyonu & LocalStorage.
Özellikler: Dinamik görev ekleme/silme, tüm listeyi temizleme ve görevler arasında gerçek zamanlı filtreleme.
Teknik: Verilerin sayfa yenilense dahi korunması için tarayıcı hafızası (Storage) kullanılmıştır.

##2. Image Search App 🔍
Odak: Fetch API & Asenkron İşlemler.
Özellikler: Unsplash API kullanarak anahtar kelimeye göre yüksek kaliteli görsel arama.
Teknik: API'den gelen JSON verilerinin dinamik kart yapılarına (Card UI) dönüştürülmesi sağlanmıştır.

##3. Currency Rates App 💹
Odak: OOP (Sınıf Yapısı) & Async/Await.
Özellikler: Güncel döviz kurları üzerinden anlık para birimi dönüştürme.
Teknik: Currency sınıfı ile API istekleri kapsüllenmiş (Encapsulation), modern asenkron yapı ile veriler yönetilmiştir.

##4. GitHub Finder App 🐙
Odak: API Zinciri & Arama Geçmişi.
Özellikler: GitHub kullanıcı profillerini ve en son repolarını görüntüleme.
Teknik: Aranan son kullanıcıların LocalStorage üzerinde bir dizi olarak saklanması ve arayüzde "Son Arananlar" olarak listelenmesi sağlanmıştır.

##5. Cinema Seat Reservation App 🎬
Odak: State Management & İndeks Yönetimi.
Özellikler: İnteraktif koltuk seçimi, film bazlı fiyatlandırma ve bilet satın alma simülasyonu.
Teknik: Koltukların indexOf ile takibi yapılarak, doluluk ve seçim statüleri kalıcı olarak kaydedilmiştir.

##6. Weather App ☁️
Odak: Manuel Promise Yönetimi.
Özellikler: Şehir bazlı anlık hava durumu, sıcaklık ve açıklama bilgisi.
Teknik: fetch istekleri özel bir Promise yapısı içinde sarmalanarak hata yönetimi (City Not Found vb.) optimize edilmiştir.

##7. Movie App 🎞️
Odak: Dinamik Template Literals & TMDB API.
Özellikler: Popüler filmleri listeleme ve film kütüphanesinde arama yapma.
Teknik: Kompleks HTML şablonları JavaScript tarafında oluşturulup DOM'a asenkron verilerle enjekte edilmiştir.

##8. Calculator App 🔢
Odak: Matematiksel Algoritmalar & Mantıksal Kontroller.
Özellikler: Dört işlem ve ondalıklı sayı desteği sunan gelişmiş hesap makinesi.
Teknik: Operatörlerin mükerrer kullanımını engelleyen static metodlar ve state takibi yapan mantıksal mimari.

##9. Password Encoder App 🔐
Odak: Veri Güvenliği & Encoding.
Özellikler: Metinleri Base64 formatında şifreleme (Encode) ve çözme (Decode).
Teknik: Tarayıcıda yerleşik btoa() ve atob() fonksiyonları ile çift yönlü veri işleme mantığı kurgulanmıştır.

##🛠️ Teknik Yetkinlikler
Bu projeler süresince aşağıdaki konularda derinlemesine pratik yapılmıştır:
Modern JS (ES6+): Classes, Static Methods, Arrow Functions, Template Literals.
Asenkron Yönetim: Fetch API, Async/Await, Promises, API Query Parameters.
Veri Yönetimi: LocalStorage entegrasyonu, State Management, Data Persistence.
UI/UX: CSS Flexbox & Grid, Responsive Design, Dynamic DOM Updates.
