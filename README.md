# 🚀 JavaScript Projects Collection

Bu depo, JavaScript dilinin temel prensiplerinden başlayarak; **Asenkron Programlama**, **Nesne Yönelimli Programlama (OOP)**, **API Entegrasyonları** ve **State Management** konularında kendimi geliştirmek amacıyla hazırladığım 9 farklı projeyi içermektedir.

---

## 📁 Proje Listesi ve Teknik Detaylar

| # | Proje | Odak Noktası | Teknik Özet |
| :--- | :--- | :--- | :--- |
| 1 | **Todo List App** 📝 | `DOM & Storage` | Dinamik görev yönetimi ve LocalStorage ile veri kalıcılığı. |
| 2 | **Image Search App** 🔍 | `Fetch API` | Unsplash API entegrasyonu ve dinamik Card UI tasarımı. |
| 3 | **Currency Rates App** 💹 | `OOP & Async` | `Class` mimarisi ile kapsüllenmiş anlık döviz çevirici. |
| 4 | **GitHub Finder App** 🐙 | `REST API` | GitHub API üzerinden profil analizi ve arama geçmişi takibi. |
| 5 | **Cinema Reservation** 🎬 | `State Management` | İndeks bazlı koltuk takibi ve biletleme mantığı. |
| 6 | **Weather App** ☁️ | `Promises` | Manuel Promise yönetimi ile şehir bazlı hava durumu takibi. |
| 7 | **Movie App** 🎞️ | `Template Literals` | TMDB API verilerinin kompleks HTML şablonlarıyla render edilmesi. |
| 8 | **Calculator App** 🔢 | `Algorithms` | Statik metodlar ve operatör kontrol algoritmaları. |
| 9 | **Password Encoder** 🔐 | `Security` | `btoa()` ve `atob()` ile Base64 veri şifreleme ve çözme. |

---

## 🛠️ Teknik Yetkinlikler

Bu projeler süresince aşağıdaki konularda derinlemesine pratik yapılmıştır:

*   **Modern JS (ES6+):** Classes, Static Methods, Arrow Functions, Template Literals.
*   **Asenkron Yönetim:** Fetch API, Async/Await, Promises, API Query Parameters.
*   **Veri Yönetimi:** LocalStorage entegrasyonu, State Management, Data Persistence.
*   **UI/UX:** CSS Flexbox & Grid, Responsive Design, Dynamic DOM Updates.
